## Beehive

### `Initialize`

- git clone 
- npm install
- npm start

### `Functionalities`


- **Authentication**: 
This functionality includes **sign up**,**sign in** and **sign out**.<br/>
User model

- **Product**: 
Includes **product listing page**(PLP) & **product details page**(PDP)<br/>
 Product model


- **Add to cart**: 
Includes add to cart functionality in **PDP**  
Order model

- **Complete order**: 
Includes complete order functionality
